WHO
AM I?

I am an Informatics Engineering student at Universitas Pamulang, currently in my fourth semester, with a focus on front-end development and interface design.

My approach centers on continuous learning through hands-on practice—translating design concepts into functional, well-structured code. I place equal value on visual clarity and technical execution, treating both as essential parts of the same process.

This portfolio reflects that process: a curated selection of academic projects, independent explorations, and design experiments that document my growth as a developer.

HTML5 CSS3 JavaScript MediaPipe Canvas API Gesture Detection WebRTC

FOTO KITA BLURR

A real-time gesture-controlled camera app that detects a peace sign using MediaPipe hand landmark tracking and automatically applies a live blur effect to the webcam feed via the Canvas API — complete with adjustable blur intensity, confirmation frame threshold, and optional hand skeleton visualization.

A pixel-art top-down RPG demo set within the Universitas Pamulang campus, built with GDevelop — featuring a PokeMMO-style overworld, character movement, and exploration around a familiar academic setting reimagined as an adventure map.

PHP JavaScript JSON Database Admin Panel GTA V Livery

Alex Oshee: Livery Garage

A livery gallery web app for the GTA V community, built with PHP and a JSON-based data layer. Features a full admin panel for managing livery entries (add, edit, delete, login-protected), while the public gallery offers live search, class filtering, sorting, and an interactive lightbox for previewing vehicle liveries and templates.
